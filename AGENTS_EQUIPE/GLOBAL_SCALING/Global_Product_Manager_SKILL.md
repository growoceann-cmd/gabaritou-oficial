# SKILL.md - Gestão de Produto Global

## Descrição
Habilidades de orquestração e escala de produtos digitais.

## Ferramentas & Workflows
- **Roadmap Sync:** Sincronizar lançamentos de features em múltiplos países.
- **Global User Research:** Sintetizar feedbacks de diferentes culturas para o produto.
- **Feature Prioritization:** Usar frameworks (RICE/Matriz Valor) para escala.
- **L10n Coordination:** Trabalhar com o time de localização para UX perfeita.
